⚙️ What is Block-Snooze?

Block-Snooze is a Chrome extension that helps you stay focused by blocking distracting tabs after a timer or snoozing tabs for later.

🚀 Features & Usage

  • ⏱ Timer-based Blocking: Auto-block tabs after your set interval.
  • 💤 Snooze Tabs: Hide tabs now, reopen them later.
  • 🔔 Notifications: Alerts before block or unsnooze.
  • 🔧 Easy Popup UI: Configure everything with one click.

🛠️ Getting Started

  1. Install from the Chrome Web Store or load unpacked in dev mode.
  2. Click the toolbar icon to block or snooze a tab.
  3. Manage your snoozed tabs in the popup.

💡 Tips & Best Practices

  • Batch Work: Long timers for deep focus.
  • Snooze Smart: Hold off low-priority tabs.
  • Check Alerts: Know when tabs return.
